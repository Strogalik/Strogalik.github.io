# TMS x Odoo — 2 КП в одном лендинге

Файлы:
- `index.html` — основная страница
- `styles.css` — стили
- `assets/` — логотип, фирменный фон и стоковые фотографии
- `sources.md` — список источников по изображениям

## Как использовать

1. Открой `index.html` в браузере.
2. Для импорта в Figma через html.to.design используй viewport `1920`.
3. Каждый `section.slide` — отдельный слайд формата 1920×1080.
4. Первые 11 слайдов — TMS.
5. Слайд 12 — разделитель новой презентации.
6. Слайды 13–18 — Odoo.

## Что лучше заменить позже

- Логотип клиента и любые реальные скриншоты интерфейса.
- Табличные и аналитические блоки на реальные данные проекта.
- Формулировки цены/состава — если появятся уточнения по этапам внедрения.

## Где менять быстро

- Тексты: `index.html`
- Цвета и размеры: `styles.css`
- Фото: заменяй файлы в `assets/` с теми же именами

## Логика визуала

- DARK-слайды: ключевой смысл, решение, цена, финальный CTA.
- LIGHT-слайды: объяснение, функционал, аналитика, перечень задач.
- Слайды чередуются так, чтобы презентация не утомляла и сохраняла ритм.
