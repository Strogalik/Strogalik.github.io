# TMS KP Landing

Одностраничный HTML-лендинг с вертикальным скроллом для коммерческого предложения ООО «ТМС».

## Как открыть
1. Распакуйте архив.
2. Откройте `index.html` в браузере.
3. Для корректного просмотра используйте окно 1920px по ширине или полноэкранный режим.

## Состав
- `index.html` — весь HTML/CSS/JS в одном файле.
- `assets/favicon.svg` — логотип TMS.

## Особенности
- Чистый HTML/CSS/JS.
- Без внешних библиотек.
- Фиксированная ширина 1920px.
- Секции высотой 1080px.
- Вертикальный скролл как у лендинга.
- Форма на последнем экране с alert-отправкой.
- Стоковые изображения подгружаются по URL из Unsplash.
