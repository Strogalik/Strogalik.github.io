# Forrest Foodtech × TMS landing

Готовый desktop-first лендинг под встречу с Forrest Foodtech.

## Что внутри
- `index.html` — вся страница в одном файле
- `assets/` — логотип TMS, фирменные фоновые элементы и временные фото-плейсхолдеры

## Как открыть
1. Скачай папку целиком.
2. Открой `index.html` в браузере.

## Что уже заложено
- структура и тексты по ТЗ
- визуальная логика Forrest-first + TMS operations layer
- шрифтовая пара Montserrat + Inter
- desktop-first сетка под 1440 px
- готовые секции под перенос в Figma
- концептуальные UI-блоки: кухня, зал, дашборд, AI, архитектура интеграции, roadmap

## Что лучше заменить своими материалами
- `assets/staff-tablet.jpg`
- `assets/chef-kitchen.jpg`
- `assets/courier-delivery.jpg`
- `assets/manager-laptop.jpg`
- при необходимости — `assets/restaurant-interior.jpg`
