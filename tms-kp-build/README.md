# TMS KP Presentation

Статичная HTML-презентация 1920×1080 с горизонтальным слайдером.

## Как открыть
1. Распакуйте архив.
2. Откройте `index.html` в браузере.
3. Для корректного просмотра разверните окно браузера на 1920×1080 или откройте в полноэкранном режиме.

## Управление
- Стрелки внизу слайда.
- Клавиши ← / →.
- Точки навигации.

## Состав
- `index.html` — весь HTML/CSS/JS в одном файле.
- `assets/favicon.svg` — логотип/иконка TMS.

Внешние изображения подключены по URL из Unsplash.
